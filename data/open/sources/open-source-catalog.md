Scriptura Lab includes a small open corpus seed for RAG testing and development.

Project Gutenberg provides the King James Version as public domain in the United States. The bundled KJV files are short excerpts intended for local retrieval testing.

The SBL Greek New Testament is available under a Creative Commons Attribution 4.0 license. The bundled SBLGNT file is a short John 1 excerpt with attribution metadata.

The Open Scriptures Hebrew Bible project provides Hebrew Bible resources, and its site notes that the WLC text remains in the public domain while lemma and morphology data use Creative Commons Attribution 4.0 terms. The bundled OSHB file is a short Genesis 1 excerpt with attribution metadata.

unfoldingWord publishes open-licensed biblical content resources under Creative Commons Attribution Share-Alike 4.0 terms. No unfoldingWord content text is bundled yet; add it only through an importer that preserves exact attribution and license metadata.
